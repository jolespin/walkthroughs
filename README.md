**File naming scheme:**

* .tsv[.gz] - (Gzipped) tab-delimited table
* .fasta,.fa,.ffn,.faa,.fna[.gz] - (Gzipped) fasta file
* .nwk[.gz] - (Gzipped) newick phylogenetic tree
